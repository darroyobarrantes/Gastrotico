Instalacion del proyecto.

1. Instalar selenium.
2. Descargar el webDriver compatible con la version de chrome del ordenador.
3. Mover el archvo del webDriver a la carpeta raiz del proyecto.




En caso de:

Al importar los modulos arroje el error "ModuleNotFoundError: No module named 'x":
-Asegurarse que se estan incluidos los path con las rutas de las subcarpetas del proyecto.

Al ejecutar el Script arroje el error "SessionNotCreatedException: session not created: This version of ChromeDriver only supports Chrome version": 
Revisar la version de google chrome y descargar el webdriver compatible con esa version.
